Meal Planner API

Meal Planner API is an API built using Java and SQL for data persistence. This API allows the user to add their favourite meals to a database as well as the ingredients required. These meals will be added to an SQL database using the Statement function to execute SQL statements.


Once the user has added their favourite meals, the API will then ask the user to select their breakfast, lunch and dinner meals for each day of the week. Once this is done the API will then create a shopping list as a .txt file including every ingredient required for every meal for the week.


Skills: Java, PostgreSQL, CRUD.


Find out more- https://github.com/kamrantahir2/MealPlanner

Project Showcase

To begin, the user inputs various meals and their ingredients which will then be added to a PostgreSQL database. Once the user is happy with the variety of meals they can then begin to plan their meals for the week.

The API will prompt the user to select their meals for breakfast, lunch and dinner for the entire week and save the results to another table in our database. The meal plan for the week will then be printed and if the user is happy they can then save their meal plan.
