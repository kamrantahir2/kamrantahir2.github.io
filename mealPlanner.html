Meal Planner API

Meal Planner API is an API built using Java and SQL for data persistence. This API allows the user to add their favourite meals to a database as well as the ingredients required. These meals will be added to an SQL database using the Statement function to execute SQL statements.


Once the user has added their favourite meals, the API will then ask the user to select their breakfast, lunch and dinner meals for each day of the week. Once this is done the API will then create a shopping list as a .txt file including every ingredient required for every meal for the week.
