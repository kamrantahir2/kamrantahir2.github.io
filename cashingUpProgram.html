Cashing Up Program

Cashing Up Program is a desktop application designed to reduce the time taken to count tills for shop owners. By asking the user to type in the quantity of their coins and notes and doing the calculation for them, the time taken to count each till is reduced by 50%.


This project is a desktop application complete with a GUI built using JavaFX and SceneBuilder.


Skills: Java, JavaFX, SceneBuilder


Find out more- https://github.com/kamrantahir2/CashingUpProgram_V2

Project Showcase

The user is asked to input the quantity of coins and notes and automates the calculations.

After the user has submitted their coins and notes, the calculation is performed, the float to be kept for the next day is removed and the total takings for the till is displayed.
